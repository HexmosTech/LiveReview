# LiveReview Assets Configuration

This file ensures that the assets folder is properly copied to the build output.

## Overview

The assets in this folder include:
- Logo files (multiple formats)
- Brand guidelines
- Example screenshots
- Favicon

## Usage

All assets should be referenced using the path `/assets/...` in the application.

For example:
```jsx
<img src="/assets/logo.svg" alt="LiveReview Logo" />
```

## Guidelines

Please refer to the guidelines folder for proper usage of brand assets.
